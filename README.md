# 🎬 Movie Search (OMDB API)

A Flask web application that searches movies using the OMDB API with a clean, responsive interface.

## 🚀 Quick Start

```bash
# Clone and setup
git clone <your-repo>
cd movie-search-app

# Install dependencies
pip install -r requirements.txt

# Get your free API key from http://www.omdbapi.com/apikey.aspx
# Create .env file with:
OMDB_API_KEY=your_api_key_here

# Run the app
python app.py

# Open http://localhost:5000
